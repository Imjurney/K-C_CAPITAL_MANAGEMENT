## 개발 목표
- 뉴질랜드 인터넷 환경에 맞춰 다양한 네트워크 환경에서 테스트를 진행합니다.
- 테스트 툴을 통해 오차범위를 최대한 좁힙니다.
- admin 계정 대신, 데이터 수정을 위해 ```data/*``` 하위목록으로 데이터를 관리합니다
- 데이터는 클라이언트 측에서 모두 관리합니다.
- entry Page만 SSR를 시도합니다.
